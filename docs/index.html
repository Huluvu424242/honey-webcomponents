Styling der Komponente

Globale Definitionen um alle Komponenten identisch zu stylen

  /* <style> set all speaker at page to defaults of component via :root */
    :root {
      --honey-webcomponents-background: transparent;
      --honey-webcomponents-color: blue;
      --honey-webcomponents-padding: 5px;
      --honey-webcomponents-font-size: medium;
      --honey-webcomponents-border: 0;
      --honey-webcomponents-width: 36px;
      --honey-webcomponents-height: 36px;
    }

Styling über css class und custom properties verändert
